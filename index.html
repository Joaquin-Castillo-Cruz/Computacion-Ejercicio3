Es la forma en que CSS estructura todos los elementos como cajas rectangulares. Entender el modelo de caja es clave para controlar el espacio entre elementos y su diseño.


Partes del modelo de caja:

  • Contenido (content): el texto o imagen dentro del elemento.
  • Relleno (padding): espacio entre el contenido y el borde.
  • Borde (border): el marco que rodea al padding.
  • Margen (margin): espacio entre el borde del elemento y otros elementos.
